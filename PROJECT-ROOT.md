## OneDrive shortcut (optional)

The folder `...\MindCET\Tandem - Claude Code` can point here via a junction.

1. Close Cursor completely.
2. Run `UNIFY-PROJECT.ps1` from that OneDrive folder (right-click → Run with PowerShell).
3. Reopen the same OneDrive folder in Cursor — it will show this full project.

Until then, open **`C:\dev\buildpilot`** directly in Cursor (File → Open Folder).
